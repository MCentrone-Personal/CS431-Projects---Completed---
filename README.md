# CS431-Projects

Contains all project file for CS-431 - Principles in Programming Languages at SUNY Polytechnic Institute.


### Directories:
- *Project1-AutomateHealing:* Contains the MacroQuest files for automating healing of a tank. This is project 1.
- *Project2-LuaHealing:* Contains the Lua scripts for project 2, which converts the MacroQuest macro to a Lua plugin.
- *Project3-NPC_Quest:* Contains the Perl script for project 3, which is a quest script for an NPC.
- *FinalProject:* Contains the final project for the class. This is a Perl quest script that is a casino dealer.
